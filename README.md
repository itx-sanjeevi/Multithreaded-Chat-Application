# Multithreaded Chat Application

A Java-based client-server chat application using sockets and multithreading.

## Features
- Multiple clients support
- Real-time communication
- Broadcast messaging
- Client join/leave notification

## Technologies Used
- Java
- Socket Programming
- Multithreading

## How to Run

### Compile
javac ChatServer.java
javac ChatClient.java

### Run Server
java ChatServer

### Run Client
java ChatClient
